# Novo Criativo

1. Pergunte para qual anúncio é o novo criativo (liste os ativos via Meta MCP).
2. Pergunte o ângulo da nova variação: **preço · prova social · urgência · brinde · transformação**.
3. Use o agente `criativo` para gerar copy + briefing de imagem.
4. **Gere a imagem com nano-banana** (backend padrão Easy4u).
5. Mostre preview em texto (descrição da imagem + copy).
6. Pergunte: "Posso subir como variação A/B no anúncio X?"
7. Só após "sim", crie o ad novo na Meta via MCP.
